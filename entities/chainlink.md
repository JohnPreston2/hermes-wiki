# Chainlink

## Overview

Réseau oracle #1 — feeds de prix off-chain vers smart contracts on-chain. Infrastructure critique pour DeFi (AAVE, Synthetix, Compound). CROSS-CHAIN INTEROPERABILITY (CCIP) pour messenger inter-chaînes.

**Lancé:** 2017 | **Token:** $LINK | **Chain:** multi-chain (Ethereum, Polygon, Avalanche, etc.)

## Faits Clés

- **Price Feeds:** 2000+ feeds de prix sur 100+ réseaux
- **CCIP:** Cross-chain messaging — utilisé par Circle, SWIFT pilota
- **Functions:** compute off-chain任意
- Revenue: payement par-protocole pour feeds
