# Morpho

## Mécanique
Lending peer-to-peer optimisé. Morpho Blue = marchés isolés permissionless (chaque marché = un couple collateral/borrow avec oracle et LLTV fixés). Pas de governance on-chain sur les marchés — le créateur définit les paramètres. MetaMorpho vaults = couche d'agrégation au-dessus.

## Risques connus
- Marchés isolés = fragmentation de liquidité. Un marché mal paramétré peut être exploité sans affecter les autres.
- Dépendance aux curateurs de vaults (Steakhouse, Gauntlet) pour l'allocation.
- Pas de Safety Module comme Aave — pertes absorbées par les déposants du marché concerné.

## Particularités
- Croissance agressive: $7.4B TVL, gagne des parts sur Aave surtout via Base (Coinbase push).
- Coinbase a intégré Morpho pour les prêts adossés à SOL — signal fort d'adoption institutionnelle.
- Architecture fondamentalement différente d'Aave: plus efficace en capital mais plus complexe à comprendre.
- Base est la chaîne dominante pour Morpho — symbiose avec l'écosystème Coinbase.
