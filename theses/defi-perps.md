# Thèse: DeFi Perps

## Conviction

## Dynamiques actuelles
- SpaceX pre-IPO sur HL = cas d'usage qui n'existe nulle part ailleurs. Attire de nouveaux utilisateurs.
- CME/ICE poussent les régulateurs contre HL — risque réglementaire le plus élevé du secteur.
- Le bridge HL ($5B TVL) est un proxy de confiance: si le TVL bridge monte, les traders sont confiants.
- Funding rate spreads HL vs Binance = alpha. Z-scores extrêmes = mean reversion trade.

## Ce qui changerait ma thèse

- Si dYdX lance un produit différenciant sur sa propre chain → possible comeback.
- Si Drift capture le volume Solana perps → duopole HL/Drift.
