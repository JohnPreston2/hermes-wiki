# Hyperliquid

## Mécanique
Perp DEX orderbook on-chain (L1 custom, anciennement Arbitrum). Pas d'AMM — carnet d'ordres central limit. Funding rates toutes les 8h. Vaults communautaires (HLP) fournissent la liquidité. Token HYPE = gas + staking + governance.

## Risques connus
- Mai 2026: CME + ICE poussent les régulateurs US à enquêter (manipulation, energy trading).
- Centralisation du validateur set — pas encore fully decentralized.
- HLP vault a pris des pertes significatives lors du squeeze JELLY (mars 2025).
- Pas de smart contracts vérifiables — code propriétaire, trust model élevé.

## Particularités
- Dominant sur les perps DEX par volume ($150M+/jour). dYdX et GMX en net recul.
- SpaceX pre-IPO tokenisé sur HL = cas d'usage unique, attire l'attention institutionnelle.
- Bitwise a lancé un fonds HL avec staking rewards — première adoption institutionnelle US.
- Le spread funding HL vs Binance est un signal contrarian fiable (z-score élevé = mean reversion).
