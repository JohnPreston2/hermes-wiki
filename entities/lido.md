# Lido

## Overview

Leader du liquid staking Ethereum. Permet de staker n'importe quel montant d'ETH tout en recevant un token liquide (stETH) utilisable dans DeFi.

**Lancé:** Dec 2020 | **TVL:** ~$21B | **ETH staké:** ~28% du total ETH

## Faits Clés

- stETH: token ERC-20 = ETH staké + rewards accumulés
- Rewards: distribués quotidiennement, compoundées automatiquement
- Node operators: 29 operators sélectionnés (Lido DAO vote)
- Withdrawals: après The Merge, withdrawals ETH enabled
- Solana: Lido sur Solana aussi (stSOL), TVL ~$200M
